# Loop    

## For Loop     
In for loop below, i will be 0 to 9.    

    for(int i = 0; i<10; i++) {
      System.out.println(i);
    }

equals to:   

    for(int i=0; i<10;)
    {
      System.out.println(i);
      i++;
    }

Note: In for loop, ++i same to i++.     

Another way for loop     

    int[] integers = {1, 2, 3, 4};
    for(int i : integers){}
