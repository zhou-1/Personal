## Question Description
Clear language for describing question.    
Given a binary tree, return the inorder traversal of its nodes' values.     

Example:     
Input: [1,null,2,3]       
---1   
----\   
-----2  
----/   
---3   
Output: [1,3,2]     

Follow up: Recursive solution is trivial, could you do it iteratively?     



----------------------------------

### Code in Java.     
I. Recursive for left, center and then right.    

    /**
    * Definition for a binary tree node.
    * public class TreeNode {
    *     int val;
    *     TreeNode left;
    *     TreeNode right;
    *     TreeNode(int x) { val = x; }
    * }
    */
    class Solution {
     public List<Integer> inorderTraversal(TreeNode root) {
        //prepare for result
        List<Integer> result = new ArrayList();
        
        //check for extreme condition
        if(root == null){
            return result;
        }
        else if(root.left == null && root.right == null){
            result.add(root.val);
            return result;
        }
        
        //recursive for root value and result list
        recursive(root, result);
        
        return result;
        
     }
    
     //recursive for root value
     public void recursive(TreeNode root, List<Integer> result){
        //Inorder Traversal: root.left, root, root.right
        if(root.left != null){
            recursive(root.left, result);
        }
        
        result.add(root.val);
        
        if(root.right != null){
            recursive(root.right, result);
        }
        
     }
    
    }


II - 1. Non recursive method with stack

    class Solution {
     public List<Integer> inorderTraversal(TreeNode root) {
        //prepare for result
        List<Integer> result = new ArrayList();
        
        //check for extreme condition
        if(root == null){
            return result;
        }
        else if(root.left == null && root.right == null){
            result.add(root.val);
            return result;
        }
        
        //non-recursive method
        
        //stack method
        Stack<TreeNode> stack = new Stack<> ();
        TreeNode cur = root;
        while(cur != null || !stack.isEmpty()){
            //keep track for left node
            while(cur != null){
                stack.push(cur);
                cur = cur.left;
            }
            
            cur = stack.pop();
            result.add(cur.val);
            cur = cur.right;
            
        }
        
        return result;
     }
   
    }
